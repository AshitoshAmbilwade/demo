# Available API Endpoints
GET – Fetch restaurant data
GET /restaurant

POST – Place an order
POST /order

PATCH – Update data
PATCH /restaurant/status
PATCH /restaurant/rating
PATCH /order

DELETE – Remove data
DELETE /menu/:itemId
DELETE /order/:orderId
